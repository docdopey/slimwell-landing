# SlimWell Landing Page

A professional healthcare SaaS landing page for weight loss clinics.

**Live Demo:** [slimwell-landing.vercel.app](https://slimwell-landing.vercel.app/)

**Features:**
- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations
- Industry-specific messaging for healthcare
- Trust signals (HIPAA, SOC 2, testimonials)
- Multi-section layout with CTAs
- Pricing tiers and features comparison

**Built with:**
- HTML5
- CSS3 (animations, gradients, flexbox/grid)
- Vanilla JavaScript

**What I learned:**
- Healthcare industry requirements and messaging
- Conversion rate optimization
- Responsive design principles
- CSS animations and transitions
